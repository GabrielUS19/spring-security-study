# Spring Security 🚀

Repositório construído com o objetivo exclusivo de estudar o ecossistema do Spring Security, servindo como base de conhecimento para projetos pessoais maiores.

## 🛠️ Tecnologias e Conceitos Aplicados
- **Spring Boot 4.x** & **Spring Security**
- **Autenticação & Autorização:** (Ex: Implementação de JWT, criptografia de senhas com BCrypt, Refresh Token Rotation.)
- **Database Migrations:** Flyway gerenciando o versionamento do banco de dados.
- **Padrões de Código:** Uso de DTOs (Data Transfer Objects) para o tráfego seguro de dados entre as camadas.
- **Arquitetura:** Divisão clara de responsabilidades (Controller, Service, Repository).

## 🎯 Propósito
Este projeto não possui regras de negócio densas porque o foco total foi mapear a fundo a cadeia de filtros (Filter Chain) do Spring Security e a persistência correta de usuários.
